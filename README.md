# Caffix (caffine22 프로젝트)

## 개요
직장인과 학생을 위한 **카페인 섭취 관리 앱**입니다.  
- 하루 카페인 누적 계산  
- 오후 2시 푸시 알람 제공 → 기록 잊지 않도록 안내  
- 카페인 정보 제공 및 프로필 관리  

## 주요 기능
- **회원가입 / 로그인**  
  - 일반 회원가입, 비밀번호 찾기, 구글 로그인 지원
- **카페/메뉴/사이즈 선택**  
  - 드롭다운 메뉴로 선택 → 카페인 함량 표시  
- **카페인 섭취 기록 및 누적**  
  - 사용자가 입력 → 하루 누적 계산 → 0시 초기화  
- **푸시 알람**  
  - 오후 2시에 입력 알림  
- **프로필 관리**  
  - 이메일 확인, 비밀번호 변경, 로그아웃  

## 사용 기술
- **언어 / IDE:** JAVA, Android Studio  
- **백엔드 / DB / 푸시:** Firebase  
- **버전 관리:** Git, GitHub  

## 팀 역할
- **본인:** 앱 개발 전담  
  - 드롭다운 메뉴, 누적 계산, 푸시 알람, 계정 관리 등  
- **팀원:** 자료 조사 및 데이터 수집  

## 프로젝트 특징
- 직장인/학생의 카페인 섭취 습관을 쉽게 관리할 수 있도록 설계  
- UI와 기능이 직관적, 실제 사용성 검증 완료  
- 데이터 기반 접근: 프랜차이즈 카페 메뉴별 카페인 함량 DB 구축
